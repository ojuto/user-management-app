# User management app

## Starting the application

1. Install [NodeJs](https://nodejs.org/en)
2. Install dependencies with: `npm install`
3. Edit the environment variables in `enviroment.ts` && `enviroment.development.ts`
4. Start the application with: `npm run zendesk-serve`
5. Open your Zendesk instance, navigate to a ticket and append `?zcli_apps=true` to the url

## Testing the application

1. Running all tests: `npm run test`
2. See the test coverage: `npm run test:coverage`
